Class OnSliderComponent

name

OnSlider

description

Este componente cria um slider. É necessário utilizar o componente on-slide

usage

<div class="on-slider-box">
  <on-slider>
    <on-slide>
      <p>Slide 1</p>
    </on-slide>
    <on-slide>
      <p>Slide 2</p>
    </on-slide>
    <on-slide>
      <p>Slide 3</p>
    </on-slide>
  </on-slider>
</div>
